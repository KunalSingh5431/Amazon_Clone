📖 Introduction
Welcome to the Amazon Clone project! This project is a simplified frontend version of the popular e-commerce platform Amazon, built to demonstrate web design and development skills using HTML and CSS.

✨ Features
Responsive Design
Product Listings
Shopping Cart UI
Search Bar UI
User Authentication Pages (Sign Up, Sign In)
Order Summary Page
🛠️ Technologies Used
HTML
CSS
